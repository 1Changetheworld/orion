ORION -- your portable AI memory.

To wake Orion on this computer, double-click the file for your OS:

  Windows  ->  Wake Orion (Windows).bat
  macOS    ->  Wake Orion (Mac).command
  Linux    ->  Wake Orion (Linux).desktop   (or run: bash install.sh)

If the drive already has an Orion brain (you've used him before),
this just wires THIS machine to wake him here. ~30 seconds, no
questions. After this, your AI tools (Claude Code, Codex CLI, Gemini
CLI, Cursor, VS Code, Claude Desktop, browser extensions) all reach
the same brain on this machine.

If the drive has no brain (first-ever Orion), the conversational
setup runs so Orion can introduce himself, ask your name + how to
address you, and choose a fuel.

Pull the USB any time and Orion goes with the drive. The host
returns to baseline. No residue.

------------------------------------------------------------------
First-time security notes per OS:

  Windows: SmartScreen may say "Windows protected your PC."
           Click "More info" then "Run anyway." Normal for any
           unsigned open-source app (Inkscape, OBS, Handbrake all
           behave the same way on first run).

  macOS:   No warnings expected. The .command extension bypasses
           Gatekeeper, and USB-borne files don't carry the
           quarantine attribute.

  Linux:   GNOME may need a right-click on Wake Orion (Linux).desktop
           -> "Allow Launching" the first time. Or just open a
           terminal in this folder and run: bash install.sh
