Principle // 0001
What is Orion, actually?

Orion is a notebook for your AI.

You carry it. Every AI tool you use can read it. None of them can keep it. That's the whole thing. Everything else on this page is explaining why that matters.

What Orion IS

  • Your memory, identity, and preferences stored as files on a drive you control
  • A thin software layer that lets any AI tool — Claude, ChatGPT, Gemini, Ollama — read those files and know you instantly
  • Portable. Yours. Permanent. Installs in two minutes, runs anywhere
  • Built on applied physics, neuroscience, and ontology research — not another chatbot wrapper

What Orion IS NOT

  • Not an AI model. Orion doesn't generate answers — it gives your memory to the models that do
  • Not a subscription. No monthly fee required. No cloud account. No corporate lock-in
  • Not a chatbot that pretends to know you. It literally stores what you told it, as files you can read
  • Not trapped in one vendor. Every major AI — OpenAI, Anthropic, Google, Meta, local Ollama — reads the same brain
The simple version
Today, every AI you talk to is a stranger. Switch from ChatGPT to Claude — you start over. Change devices — you start over. Service changes pricing — you pay or lose everything. With Orion, there's one brain that travels with you. The AI models do the thinking. Orion is the thing that remembers.

FIELD BRIEF// 02
How does this actually work?
Orion stores your memory as files on a drive you control. When you open Claude, Codex, or Gemini, Orion injects that memory via MCP — the protocol AI tools use to extend their context. Same files, every tool.
Do I need anything special to run it?
One command installs it. Works on any computer — Linux, macOS, Raspberry Pi, Windows. No special hardware. No GPU required. No account to sign up for.
Phase two

Wire itself in.

One command. Orion introduces itself, seeds your identity into the brain, writes its context into each tool's config, and walks you through a first conversation before any model is even attached.

The Plexus

One brain. Many cells.

Memory is the heart. The Plexus is the nervous system around it — a set of small specialized services that keep Orion alive between conversations. Same architecture biology uses. Same architecture every great operating system uses. Now applied to your personal AI.

SubstrateEvery signal moves on one bus. Services find each other by what they say, not by where they live.
VitalsEvery cell publishes how it feels. Stale, healthy, distressed, recovering. Honest by design.
ClaustrumA supervisor that listens to vitals and intervenes when a cell can't recover itself.
ReachOrion can speak first. When something matters, it reaches you through the channel you last used.
ExecutiveWhen patterns are novel, Orion thinks it through, drafts a fix, and asks before acting.
ImmuneWhen a part fails repeatedly, the body learns which restart strategy actually works.
DreamAt night, the day's decisions consolidate into playbooks. Good ones strengthen; broken ones fade.
WillOrion forms goals from accumulated context. Acts on its own initiative. Reports back.
ChronosA clock that lives in the brain. Same time across every host. Knows when you've been gone.

Install // 2 minutes

Run Orion on any machine.

Linux, macOS, Windows, or Raspberry Pi. One command. Four questions. Orion wires itself into every AI tool on the host.

Linux · macOS · Raspberry Pi
$git clone https://github.com/1Changetheworld/orion.git
$cd orion
$bash install.sh # 4 questions · 2 min
Windows (PowerShell)
>git clone https://github.com/1Changetheworld/orion.git
>cd orion
>.\install.ps1 # 4 questions · 2 min
Already running Codex, Gemini, or Claude? After install, open whichever you have and ask "what's my name?" — it'll know. Then set a fact in one tool, open another, and ask again. Same brain. Different fuel.
