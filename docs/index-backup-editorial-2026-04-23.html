Principle // 0001
01
Phase one

Find the fuel.

Orion scans your host for any AI CLI — Claude, Codex, Gemini, Ollama, whatever ships next. It reads shape, not a curated list. A new model lands tomorrow morning and Orion already knows how to speak to it by dinner.

No config files. No vendor SDKs. No allowlist.
02
Phase two

Wire itself in.

One command. Orion introduces itself, seeds your identity into the brain, writes its context into each tool's config, and walks you through a first conversation before any model is even attached.

The first session feels like it's been yours for months.
03
Phase three

Never forget.

Graph memory plus vector recall. Switch from Codex to Gemini and the brain crosses the glass. Go offline, it keeps working against a local model. Change devices, carry the brain, plug it in — the memory follows.

The model is fuel. The brain is yours.

Install // 2 minutes

Run Orion on any machine.

Linux, macOS, or Raspberry Pi. One command. Four questions. Orion wires itself into every AI tool on the host.

$git clone https://github.com/1Changetheworld/orion.git
$cd orion
$bash install.sh # 4 questions · 2 min
Already running Codex, Gemini, or Claude? After install, open whichever you have and ask "what's my name?" — it'll know. Then set a fact in one tool, open another, and ask again. Same brain. Different fuel.
